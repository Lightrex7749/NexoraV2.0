# Nexora Database

This folder contains a MongoDB/Mongoose backend scaffold for the Nexora startup platform.

## Structure
- models/: Mongoose schema definitions
- docs/: design and deployment documentation
- sample-data/: seed-ready JSON examples
- seed/: scripts for populating data
- config/: database connection settings

## Setup
1. Update the MongoDB URI in .env
2. Install dependencies with npm install
3. Run npm run seed
